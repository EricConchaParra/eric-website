Instalar PMToolkit BETA

Sigue estos sencillos pasos para instalar la extensión en tu navegador Chrome (o navegadores basados en Chromium como Brave o Edge).

Descargar PMToolKit.zip Ver repositorio en GitHub

Si quieres revisar el código fuente, releases o documentación técnica, el repositorio público está disponible en GitHub.

Pasos de Instalación

  1. Descomprimir y guardar en una carpeta segura.
  2. Abrir chrome://extensions/ en Chrome.
  3. Activar el Modo Desarrollador (switch arriba a la derecha).
  4. Clic en "Cargar Extensión sin empaquetar".
  5. Seleccionar la carpeta recién descomprimida.
  6. ¡Listo! la extensión ya está activada.
